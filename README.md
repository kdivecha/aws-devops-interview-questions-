## AWS + DevOps Interview Questions (120)

This folder contains a curated question bank for AWS and DevOps interviews, with short answers and concise explanations.

### How to use
1. Attempt the question without looking at the answer.
2. Check the relevant topic file (`AWS_QUESTIONS.md` or `DEVOPS_QUESTIONS.md`) for the provided answer + brief rationale.
3. Re-attempt after a short break to improve recall.

### Content breakdown
- AWS: IAM/STS, VPC & networking, compute, storage, databases, serverless, security/compliance
- DevOps: CI/CD & GitOps, Terraform/IaC, containers & Kubernetes, monitoring/observability, reliability/SRE, security/supply-chain, incident response

### File layout
- `AWS_QUESTIONS.md`: Questions 1-65 (AWS)
- `DEVOPS_QUESTIONS.md`: Questions 66-120 (DevOps)
